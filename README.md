## INSTALLATION 

For Linux x86_64, Get binary in release page 


## BUILD 

For other platforms, compile from source

```s
git clone https://github.com/gaetan1903/ping_stat/
```

```
go mod tidy
```

```
go build
```

```
./ping-stat
```
